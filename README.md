# VolleyballRotation

排球裁判员辅助微信小程序（本地版）。

## 功能概览

- 团队机制：输入 6 位裁判团队编号创建/加入同一场比赛
- 创建流程：仅在“房间设置页”保存成功后才真正创建团队，避免占号
- 密码机制：创建团队与加入团队均使用 6 位数字密码
- 双裁判协作：同房间内实时看到一致的比分、站位与轮转
- 房间设置：
  - 赛制选框：`5局3胜` / `3局2胜` / `1局1胜(15分)` / `1局1胜(25分)`
  - 内置得分规则：多局赛前几局 25 分、决胜局 15 分；一局赛按所选 15/25 分
  - A/B 队名称
  - A/B 队颜色（内置 12 色，A/B 不能同色）
- 首发配置：I-VI + L1/L2 球员号码输入
  - 主力位必填
  - 同队号码不可重复（失焦即时提示）
- 比赛页：
  - 顶部固定布局：左侧 X 区、中间比分区、右侧 Y 区
  - 在线人数显示
  - 发球方展示、发球位高亮与动画
  - 手动轮转 + 得分触发轮转（统一走同一套轮转逻辑）
  - 比分撤回（可连续撤回）、左右换边、计分记录
  - 比赛结束后禁止轮转；撤回到进行中后自动恢复
  - 决胜局自动换边：`5局3胜` 和 `3局2胜` 在决胜局先到 8 分自动换边
  - 单局结束与整场结束判定、状态展示（右上角显示“已结束 x:y”）
- 安全区适配：横屏比赛页按 `safeArea` 动态避让刘海/灵动岛

## 项目结构

- `miniprogram/pages/create-room`：首页（创建/加入房间）
- `miniprogram/pages/room`：房间设置与首发配置
- `miniprogram/pages/match`：比赛主界面
- `miniprogram/utils/room-service.ts`：房间与比赛状态本地存储服务
